## Overview
This skill is a simple example that demonstrates how to create a skill using the Skill format and App Actions in Pulse Editor. The skill includes an App Action that can be called by AI agents to perform a specific task. This example serves as a starting point for developers who want to create their own skills for Pulse Editor.


## What this skill does
This skill defines an App Action called `exampleSkill` that takes two arguments: `arg1` (a string) and `arg2` (a number, optional). The action returns an object containing two results: `result1` and `result2`, which are strings that echo the input arguments.
